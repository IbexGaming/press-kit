# IGL Press Kit
A compact guide to the included brand assets and where to use them.

## Quick use
- Pick a banner that matches the target platform aspect ratio.
- Use tier icons for standings, announcements, and overlays.
- Do not stretch or skew assets; scale proportionally.

## File tree
```
IGL Press Kit/                     
    igl-icon-color.png                        # IGL icon (color)
    igl-icon-white.png                        # IGL icon (white)

    stream/
        community-stream-banner.png           # Official banner for comunity streams, minimum 6.9% height of view (ca. 75px on 1080p)
```

## Usage notes
- Keep a clear margin around logos and icons.
- Favor high-contrast backgrounds for readability.
- Logo/Icon Exports are at 2x incase the destination supports retina/high-DPI.
